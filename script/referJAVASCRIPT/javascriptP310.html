01. 브라우저 객체 : window 객체 메서드 : window.innerWidth() / window.innerHeight() : 브라우저 가로/세로 값

클릭 : 브라우저 화면의 가로 값을 가져옵니다. 클릭 : 브라우저 화면의 세로 값을 가져옵니다. 클릭 : 브라우저 전체의 가로 값을 가져옵니다. 클릭 : 브라우저 전체의 가로 값을 가져옵니다. 클릭 : 브라우저 위쪽 위치 값을 가져옵니다. 클릭 : 브라우저 왼쪽 위치 값을 가져옵니다. 
{
    const btn1 = document.querySelector(".btn4-1");
    const btn2 = document.querySelector(".btn4-2");

    const width = window.innerWidth;
    const height = window.innerHeight;

    btn1.addEventListener("click", () => {
        document.querySelector(".result4-1").innerHTML = width + "px";
    })
    btn2.addEventListener("click", () => {
        document.querySelector(".result4-2").innerHTML = height + "px";
    })

    window.addEventListener("resize", () => {
        const width = window.innerWidth;
        const height = window.innerHeight;

        document.querySelector(".result4-1").innerHTML = `이 브라우저의 가로 값은 ${width}px 입니다.`;
        document.querySelector(".result4-2").innerHTML = `이 브라우저의 가로 값은 ${height}px 입니다.`;
    })
}
